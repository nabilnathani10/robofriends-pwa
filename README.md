# robofriends-pwa
Udemy tutorial - adding PWA capabilities to a react app

To run the project:

1. Clone this repo
2. Run `npm install`
3. Run `npm start`
4. To test: `npm test`



Testing using Enzyme
https://airbnb.io/enzyme/docs/api/

Snapshot Testing
https://jestjs.io/docs/en/snapshot-testing

Code Coverage
npm test -- --coverage
